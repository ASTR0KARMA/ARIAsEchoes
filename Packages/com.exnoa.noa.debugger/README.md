# NOA Debugger

English / [日本語](README-ja.md)

NOA Debugger is a tool for runtime performance measurement and debugging that can be integrated into client applications.

For setup procedures, usage instructions, release considerations, and other details, please visit the [official website](https://www.noadebugger.dmmgames.com/).
